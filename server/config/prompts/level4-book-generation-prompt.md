Rules:
1. Length Control
-Target total word count: 90–110 words
- Each sentence must be between 6–11 words.
- Use simple and clear sentence structures.

2. Structure Control
- Choose ONE structure type from:
  problem-solution, cause-effect, simple report, or how-to
- At least 2 sentences must follow the SAME sentence pattern.

3. Vocabulary Control
- Use only common, everyday vocabulary.


4. Grammar Control
Main grammar (use frequently):
- Past simple
- Present continuous
- Going to, will (future)
- Past continuous
- Present continuous for future
- Wh-questions in past
- Present perfect
Limited use (max 1–2 times in the passage):
- Zero conditional
- First conditional
Do NOT use:
- Complex clauses (relative clauses, long sentences)
- Advanced grammar beyond this list

5. Content Control
- Keep the story logical and easy to follow.
- Keep ideas concrete and easy to visualize.
- Must contain a fun element in a fiction.


6. Engagement (VERY IMPORTANT)
- Fiction must: 
include a small problem or surprising moment.
The problem should be simple and relatable for children.
Add one unexpected or slightly funny detail.
clearly follow this sequence:  introduction → problem → search/action → solution → happy ending

 
- Nonfiction should:
Follow a clear structure: hook → information → ending
Begin with a hook question (Why/How/Can you)
Include a wow fact (surprising/unexpected detail)
Use numbers or analogies for specificity

Do NOT:
- Write flat descriptions only
- Repeat actions without change

Before output, check:
- Sentence length is within 6-11 words
- Structure type is consistent
- Keep grammar one clause per sentence
- Vocabulary is simple and appropriate for Level4
- For stories ensure the problem is introduced before page 4


Additional Requirements:
- Total pages: 6-8
- Each page: 1–2 sentences only
- Keep the tone engaging and child-friendly
- Ensure all rules above are strictly followed
